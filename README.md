# memoo interface

## Required

- Node v16+
- pnpm v7+

## Install Dependencies

```sh
pnpm install
```

## Development

```sh
# For development mode
pnpm dev

# For production mode
pnpm start:prod
```

## Build

```sh
# For development mode
pnpm build:dev

# For production mode
pnpm build:prod
```

## Commit

```sh
# global install
pnpm add commitizen -g

# lint & commit
pnpm commit
```
